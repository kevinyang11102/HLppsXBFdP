# 图书管理系统 java20

## 项目概述

图书管理系统是一个基于Java Web的项目，它运用SSM和Spring Boot后端框架，并采用Vue 2构建前端管理界面，用户端界面则使用简洁的HTML。项目高效整合了MyBatis Plus和MySQL数据库，以实现数据交互与管理的流畅性和高效性。

## 技术栈

### 前端技术

- Vue 2

### 后端技术

- Java（SSM+Spring Boot）
- 开发工具：IDEA

### 数据库

- MySQL

## 功能模块

### 系统角色

- 管理员
  - 用户信息管理
  - 图书分类维护
  - 图书信息录入与查询
  - 图书借阅及归还审批流程
  - 罚金缴纳管理
  - 留言板内容监管

- 用户
  - 图书详情浏览
  - 图书借阅与归还操作
  - 在线缴纳罚金
  - 个人收藏夹管理
